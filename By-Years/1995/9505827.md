9505827 JONES This project will finish some essential modifications to the
Research Vessel WECOMA, an NSF-owned ship operated by the Oregon State
University. Funds will be used to replace a worn out, 18 year old boiler for
which parts have been difficult to obtain, install an additional air
conditioning chiller, reduce humidity in the platform deck accommodations and
install an emergence fire fighting pumpwith remote starting capability. ***

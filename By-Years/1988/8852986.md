This project supports undergraduate education in artificial intelligence an
robotics at this institution. A high quality, modern course is offered in a
liberal arts context so that students become knowledgable in a field that is
critical to the nation's manufacturing systems. The equipment that supports this
project includes: a high-speed dedicated computer system with image digitizer,
TV camera, robot arm and expert systems and robotics software. This award is
being matched by an equal sum from the grantee.
